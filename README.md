# OpenAI GTM Operating Map

Standalone site for OpenAI GTM research artifacts — operating map, stack architecture, and top challenges.

**Live site:** https://gmehta.github.io/OpenAI/

## Tabs

1. **Operating map** — clubbed motion model: ~20 segment × lifecycle templates (5 customer segments × 4 lifecycle stages) vs. the exploding instance surface (surfaces × users × state)
2. **GTM Stack** — Interactive architecture diagram (Cytoscape.js) from draw.io source; nodes are confidence-tagged (confirmed vs. reported / inferred)
3. **Top 5 challenges** — analysis of the GTM blockers most relevant to the role

## Tech

Static HTML/CSS/JS hosted on GitHub Pages. No build step.

## Note

Reconstructed from public reporting, vendor case studies, and interviews — not an official OpenAI artifact. Base figures (weekly active users, paying subscribers, business customers) are sourced; the segment/stage/template model and any "decision surface" estimate are an analyst's model, not OpenAI-published metrics.
